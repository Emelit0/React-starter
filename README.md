## This React Starter contains

* React
* Typescript
* Babel Loader
* ESlint
* Prettier
* husky
* Sass Loader with Dart Sass
* HTMLWebPlugin
* WebpackDevServer
* Asset Resource Loader
* Source Maps
* Caching
* basic file structure


## Download the starter kit

npx degit https://github.com/Emelit0/React-starter my-app

## Install dependencies

cd my-app &&
yarn

## Run dev server

yarn start

## Build

yarn build
